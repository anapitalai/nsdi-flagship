# MinIO Integration

## Overview

Add MinIO-based object storage integration for file uploads, downloads, and asset storage.

## Requirements

- Use MinIO as the primary object store for uploaded files
- Store object URLs or object keys in the asset model
- Support upload, download, and delete flows
- Keep storage credentials server-side only
- Document any bucket naming, folder structure, and lifecycle rules here
